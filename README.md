# Henry J. Webster Personal Website

- Grab/process data with [nushell](https://www.nushell.sh/) 
- Generate HTML files with [gomplate](https://gomplate.ca/)
- Handle incremental updates with [GNU Make](https://gomplate.ca/)

- [Nix flake](https://wiki.nixos.org/wiki/Flakes) as dev env
- Deployed on [docker](https://www.docker.com/)
- Hosted on [fly.io](https://fly.io/)
