## 📋 Objectives

🏠 Create a home base for all my work

👐 Hands-free deployment from SCM to the cloud

🧪 Test-driven development and checks upon PRs

📱 Mobile-ready

🧍 Design for accessibility

🦶 Performant, with a small footprint
